# MedTracker

A simple Next.js app to help sons track medications for their elderly parents.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Simple landing page
- Dashboard with onboarding form
- Two-step onboarding process:
  1. User information (name, phone, email)
  2. Grandparent information (name, phone, medication, time to call, gender)
